## 技术架构

### 核心模块（14个）
```
mdflow/
├── cli.py                     # CLI 入口 ⭐
├── pyproject.toml             # 包配置 ⭐
├── core/
│   ├── models.py              # 数据模型
│   ├── parser.py              # 工作流解析
│   ├── structured_compiler.py # 工作流编译
│   ├── validation.py          # 工作流验证 ⭐
│   ├── post_processor.py      # 后处理层 ⭐⭐
│   ├── logging_config.py      # 日志系统 ⭐
│   ├── config.py              # 配置管理 ⭐
│   └── tools.py               # 工具注册
├── workflows/
│   └── workflow_generator/    # 元工作流
└── tests/                     # 40+ 测试用例
```

### 数据流（完整）
```
用户输入 (CLI 或 API)
    ↓
Intent Parser Agent
    ↓
意图分析
    ↓
Skeleton Builder Agent
    ↓
工作流骨架
    ↓
Agent Builder Agent
    ↓
Agent 配置
    ↓
🔧 后处理层 (自动修正) ⭐⭐
    ↓
✅ 格式正确的可执行工作流
    ↓
📋 验证系统 (可选)
    ↓
▶️  执行引擎
```

## 使用示例

### CLI 命令
```bash
# 验证工作流
mdflow validate workflows/my_workflow/

# 规范化 LLM 输出
mdflow normalize input.md -o output.md

# 生成工作流（框架）
mdflow generate "Create code review workflow" -o output/

# 执行工作流（框架）
mdflow run workflows/my_workflow/ --session test_001

# 查看帮助
mdflow --help

# 查看版本
md --version
```

### Python API
```python
from core.post_processor import normalize_workflow_output
from core.validation import validate_workflow

# 规范化输出
result = normalize_workflow_output(llm_output)
print(result.content)

# 验证工作流
errors = validate_workflow("workflows/my_workflow/")
```
