## 核心发现

### 问题根源
最初的测试失败不是因为 agent prompt 不够好，而是因为**测试用例的 prompt 不够明确**。

### 解决方案
在测试提示中添加：
```markdown
✅ 正确示例：
validate --> review : [passed]
validate --> end : [failed]

❌ 错误示例：
validate --> review : 验证通过  ❌ 不能用中文
validate --> review : passed    ❌ 缺少方括号
```

### 效果
- **之前**: LLM 使用中文条件 "验证通过"、"性能提升<10%"
- **之后**: LLM 正确使用 `[passed]`、`[failed]`、`[confirmed]` 等方括号格式
