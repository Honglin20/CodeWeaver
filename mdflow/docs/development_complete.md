
## 关键技术决策

### 1. 后处理层 vs 更好的 Prompt
**决策**: 实施后处理层

**理由**:
- LLM 输出不可能 100% 可控
- 后处理层提供确定性保证
- 可以处理各种格式变体
- 降低对 LLM 的依赖

### 2. 测试提示 vs Agent Prompt
**发现**: 测试提示中的明确示例比优化 agent prompt 更有效

**应用**:
- 在测试中展示正确/错误示例
- 格式正确率从 0% → 100%

### 3. 验证时机
**决策**: 预执行验证 + 后处理规范化

**优势**:
- 早期发现问题
- 自动修正常见错误
- 提供详细错误消息
