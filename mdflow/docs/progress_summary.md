## 环境设置建议

### 创建虚拟环境
```bash
cd mdflow
python3 -m venv venv
source venv/bin/activate  # macOS/Linux
# 或
venv\Scripts\activate  # Windows
```

### 安装依赖
```bash
pip install --upgrade pip
pip install -r requirements.txt
```

### 运行测试
```bash
# 单元测试
pytest tests/ -v

# E2E 测试（需要 API 密钥）
python test_e2e_generation.py

# 验证测试
python test_validation.py
```

### 配置环境变量
```bash
export CODEWEAVER_API_KEY="your-api-key"
export CODEWEAVER_API_BASE="https://api.deepseek.com"
export CODEWEAVER_MODEL="deepseek-chat"
export CODEWEAVER_LOG_LEVEL="DEBUG"
```
